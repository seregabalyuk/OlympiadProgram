# Основы
Изначальная программа выглядит так

```c++
#include <iostream>

using namespace std;

int main() {
    cout << "Hello world!";
}
```

Основной код пишется в **main**

```c++
int main() {
    // тут ваш код
}
```

Каждая отдельная команда отделяется **;**

```c++
int main() {
    command;
}
```

## Команды

1. создать новую переменную *(структуру/class)*
```
тип_переменной название;
```

Пример:
```c++
int;
```
